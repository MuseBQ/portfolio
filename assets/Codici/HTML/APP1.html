Building Hybrid Apps with Ionic React and Appflow

1. Create a New Ionic React Project

Start by creating a new Ionic React application:

ionic start myApp tabs --type=react --capacitor
cd myApp

2. Add Native Platforms

Add Android and iOS platforms using Capacitor:

ionic build
npx cap add android
npx cap add ios

3. Initialize Appflow

Set up your project for Ionic Appflow:

ionic integration enable capacitor
ionic link

Follow the prompts to connect your app to Appflow.

4. Project Structure

Your Ionic React project will have the following structure:

src/ ├── components/ ├── pages/ │ ├── Tab1.tsx │ ├── Tab2.tsx │ └── Tab3.tsx ├── services/ ├── App.tsx ├── main.tsx └── ... capacitor.config.ts ionic.config.json

5. Example Page Component

Here's a simple example of an Ionic React page component:

import { IonContent, IonHeader, IonPage, IonTitle, IonToolbar, IonButton } from '@ionic/react';
import './Tab1.css';

const Tab1: React.FC = () => {
  return (
    <IonPage>
      <IonHeader>
        <IonToolbar>
          <IonTitle>Tab 1</IonTitle>
        </IonToolbar>
      <IonHeader>
      <IonContent fullscreen>
        <div className="container">
          <h1>Welcome to My App</h1>
          <IonButton expand="block" onClick={() => console.log('Button clicked!')}>
            Click Me
          </IonButton>
        </div>
      </IonContent>
    </IonPage>
  );
};

export default Tab1;

6. Appflow Configuration

Configure your ionic.config.json for Appflow:

{
  "name": "myApp",
  "integrations": {
    "capacitor": {}
  },
  "type": "react",
  "id": "your-appflow-app-id"
}

7. Building for Production

Build your app and sync with native platforms:

npm run build
npx cap sync

8. Deploying with Appflow

Commit your changes and push to your connected Git repository. Then use the Appflow dashboard to:

  1. Create native builds for Android and iOS
  2. Deploy live updates
  3. Manage your app versions

Important Notes

  • For iOS builds, you need a Mac with Xcode installed
  • You'll need to set up signing certificates for both platforms
  • Appflow allows you to build iOS apps without a Mac
  • Use Ionic's UI components for a native look and feel on both platforms

9. Next Steps
